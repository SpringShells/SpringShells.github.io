# Java语言简介

Java是一门面向对象编程语言，不仅吸收了C++语言的各种优点，还摒弃了C++里难以理解的多继承、指针等概念，因此Java语言具有功能强大和简单易用两个特征。Java语言作为静态面向对象编程语言的代表，极好地实现了面向对象理论，允许程序员以优雅的思维方式进行复杂的编程。 Java具有简单性、面向对象、分布式、健壮性、安全性、平台独立与可移植性、多线程、动态性等特点。Java可以编写桌面应用程序、Web应用程序、分布式系统和嵌入式系统应用程序等。

# Java版本

  1. JavaSE(J2SE)(Java2 Platform Standard Edition)
  2. JavaEE(J2EE)(Java 2 Platform,Enterprise Edition)
  3. JavaME(J2ME)(Java 2 Platform Micro Edition)

# 程序解释——注释

在程序中，由/* */两个符号包括起来的部份是程序的注释。写注释能为其他人（包括自已）更容易明白你写的程序。也就是说程序是写给人看的，不是写给计算机看的。注释可以单独写一行或者也可以写多行。/**/不会被编译器所编译。

  1. 单行注释，以双“//”开头，简单实用
  2. 多行注释，以“/*”开头，“*/”结束，可以跨多行
  3. 文档注释，长“/**”开头，“*/”结束，可以跨多行，可以生成Html文档，命令javap
	//单行注释
	
    /*
    多行注释
    多行注释
    */
   
    /**
     * 文档注释
     */

# 基本数据类型

Java中有 8 种基本数据类型，分别为：

  1. 数字类型 ：byte、short、int、long、float、double
  2. 字符类型：char
  3. 布尔型：boolean

注意

  1. 使用 long 类型的数据要在数值后面加上 L，不要写小写 l ，因为程序里 l 和 1 容易弄错 。
  2. 使用 float 类型的数据要在数值后面加上 F 或者 f 。
  3. 使用 double 类型的数据要在数值后面加上 D 或者 d 。

# 标识符

在计算机编程语言中，标识符是用户编程时使用的名字，用于给变量、常量、函数、语句块等命名，以建立起名称与使用之间的关系。 标识符命名规则

  1. 标识符必须以字母、下划线_ 、美元符$开头。
  2. Java 标识符大小写敏感，且长度无限制。
  3. 不可以是Java的关键字和保留字。
  4. 符号常量名全部使用大写字母，大写字母之间用下划线隔开。
  5. 类名首字母大写开头
  6. 包名全部小写
  7. 方法名以小写字母开头

WARNING

类名使用 UpperCamelCase 风格，必须遵从驼峰形式。

  1. 方法名、参数名、成员变量、局部变量都统一使用 lowerCamelCase 风格，必须遵从 驼峰形式。【强制】代码中的命名均不能以下划线或美元符号开始，也不能以下划线或美元符号结束。【强制】代码中的命名严禁使用拼音与英文混合的方式，更不允许直接使用中文的方式。
  2. 常量命名全部大写，单词间用下划线隔开，力求语义表达完整清楚，不要嫌名字长。
  3. 包名统一使用小写，点分隔符之间有且仅有一个自然语义的英语单词。包名统一使 用单数形式，但是类名如果有复数含义，类名可以使用复数形式。

# 变量及赋值

Java编程语言定义了以下种类的变量： 实例变量（非静态字段），类变量（静态字段），局部变量， 参数

在 java 中用户可以通过指定数据类型和标识符来声明变量，其基本语法如下所示： DataType identifier; 或者 DataType identifier1,identifier2,identifier3...; 变量所属的类型位于前面，随后是变量名。

# 分支结构之简单if语句

# 分支结构之简单if-else语句

if语句用于条件判定，语句执行时，会先计算条件表达式的值，如果值为真，则执行语句块1。如果真为假执行语句块2。

语法：

if (表达式) {
    语句块1;
} else {
    语句块2;
}

public class IfElseDemo{
    public static void main(String[] args){
        int studentGrade=54;
        if (studentGrade>=60){
            System.out.println("Passed");
        }else{
            System.out.println("Failed");
        }
    }
}

# 分支结构之多重if-else语句

这种if语句用于多路条件判定。其中的各个表达式将会进行运算，如果条件表达式的值为真，则执行生相应语句块。 最后的else是当以上条件都不成立的情况下默认执行语句块。

语法：

if(表达式1) {
    语句块1;
} else if(表达式2) {
    语句块2;
...
} else if(表达式n) {
    语句块n;
} else {
    语句块n+1;
}

public class MutilIfElseDemo {
    public static void main(String[] args) {
        String choice = "sunny";
        if (choice.equals("sunny")) {
            System.out.println("sunny");
        } else if (choice.equals("sunny")) {
            System.out.println("rainy");
        } else if (choice.equals("sunny")) {
            System.out.println("snowing");
        } else if (choice.equals("sunny")) {
            System.out.println("overcast");
        } else {
            System.out.println("sunny");
        }
    }
}

# 分支结构之swicth语句

有时侯当我们有多个分支选择时，可以使用if else ... else if语句来写，但有时侯使用switch更方便。

语法：

switch(表达式) {
    case 值1:
        语句块1;
        break;
    case 值2:
        语句块2;
        break;
    …
    case 值n:
        语句块n;
        break;
    default:
        语句块n+1;
    break;
}

public class SwitchDemo {
    public static void main(String[] args) {
        var choice = "sunny";
        switch (choice) {
            case "sunny":
                System.out.println("sunny");
                break;
            case "rainy":
                System.out.println("rainy");
                break;
            case "snowing":
                System.out.println("snowing");
                break;
            case "overcast":
                System.out.println("overcast");
                break;
            default:
                System.out.println("sunny");
        }
    }
}

# 三目运算符

执行过程是：先判断表达式1的值是否为真，如果是真的话执行表达式2；如果是假的话执行表达式3。可以用来替代 if...else 语句。

语法：

表达式1 ? 表达式2 : 表达式3;

# 循环结构之while循环

程序先执行条件表达式，如果值为真，继续执行语句块。然后保持继续执行条件表式与语句块，直到条件表达式为假，终止循环。

语法：

while(条件表达式) {
	语句块;
}

# 循环结构之do-while循环

程序先执行条件表达式，如果值为真，继续执行语句块。然后保持继续执行条件表式与语句块，直到条件表达式为假，终止循环。

语法：

while(条件表达式) {
	语句块;
}

# 循环结构之for循环

for也和while一样能够执行循环体。

语法：

for(条件表达式1;条件表达式2;条件表达式3) {
	语句块;
}

public class ForDemo{
    public static void main(String[] args){
        for(int i=1;i<=100;i++){
            System.out.println(i);
        }
    }
}

# 结束语句之break语句

break能中断循环结构，而执行整个循环语句后面的代码。当中断条件满足时，会终止循环。如果内嵌循环内中断，则中断内嵌循环体。

# 结束语句之continue语句

continue也能中断循环结构，但它和break不同的是，contine只跳过本次循环，继续执行后面的循环。如果内嵌循环内中断，则只跳过内嵌循环体本次循环。
