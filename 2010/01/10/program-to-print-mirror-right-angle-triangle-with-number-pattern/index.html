C语言练习-用C程序打印数字右直角三角形

SpringSongs   in China
2010-01-10
  • C语言练习

用10*10网格 第一行9个空格(10-1行号)和一个数字(1行号) 第二行8个空格(10-2行号)和两个数字(2行号) 第三行7个空格(10-3行号)和三个数字(3行号) 第四行6个空格(10-4行号)和四个数字(4行号) 以此类推… 从规律来看，每行打印的都与行号有关

/*
 ============================================================================
 Name        : main.c
 Author      : SpringSongs
 Version     : 0.0.1
 Description : 用C程序打印数字右直角三角形, Ansi-style
 ============================================================================
 打印如下：
         1
        12
       123
      1234
     12345
    123456
   1234567
  12345678
 123456789
12345678910
 */

#include <stdio.h>
int main()
{
	int i;
	int j;
	for (i = 1; i <= 10; i++)
	{
		for (j = 0; j < 10 - i; j++)
		{
			printf(" ");
		}
		for (j = 1; j <= i; j++)
		{
			printf("%d", j);
		}
		printf("\n");
	}
	return 0;
}
