C语言练习-C程序用星形图案打印3

SpringSongs   in China
Wed Jan 27 2010
  • C语言练习

用11*10网格 如果行号等于1或者6或者11打印星号 如果等于10打星号 其他打印空格

/*
 ============================================================================
 Name        : main.c
 Author      : SpringSongs
 Version     : 0.0.1
 Description : 程序用星形图案打印3, Ansi-style
 ============================================================================
 打印如下：
**********
         *
         *
         *
         *
**********
         *
         *
         *
         *
**********
 */

#include <stdio.h>
int main()
{
    int i;
    int j;
    int middle = 6;
    for (i = 1; i <= 11; i++)
    {
        for (j = 1; j <= 10; j++)
        {
            if (i == 1 || i == middle || i == 11)
            {
                printf("*");
            }
            else if (j == 10)
            {
                printf("*");
            }
            else
            {
                printf(" ");
            }
        }
        printf("\n");
    }
    return 0;
}
