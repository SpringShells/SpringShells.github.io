C语言练习-C程序用星形图案打印P

SpringSongs   in China
Sat Feb 20 2010
  • C语言练习

用11*10网格 如果行号等于1或者6打印星号 如果列号等于1打印星号 如果列号等于10且行号小于6打印星号 其他打印空格

/*
 ============================================================================
 Name        : main.c
 Author      : SpringSongs
 Version     : 0.0.1
 Description : 程序用星形图案打印P, Ansi-style
 ============================================================================
 打印如下：
**********
*        *
*        *
*        *
*        *
**********
*         
*         
*         
*         

 */

#include <stdio.h>
int main()
{
    int i;
    int j;
    for (i = 1; i <= 10; i++)
    {
        for (j = 1; j <= 10; j++)
        {
            if (i == 1 || i == 6)
            {
                printf("*");
            }
            else if (j == 1)
            {
                printf("*");
            }
            else if (j == 10 && i < 6)
            {
                printf("*");
            }
            else
            {
                printf(" ");
            }
        }
        printf("\n");
    }
    return 0;
}
